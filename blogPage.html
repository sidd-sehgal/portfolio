How to make an Analog Clock using JavaScript, HTML and CSS by Bit Coders

Today we'll learn how to make an Analog clock using JavaScript, HTML
and CSS step by step with Source Code!!

You can use this analog clock in your website to enhance the design and
make it look more beautiful to the users.

You can impress you friends too.

Requirements:

  • Computer or Laptop.
  • Basic knowledge of JavaScript, HTML and CSS.
  • Code Editor (such as VS Code, Atom, Notepad++, etc.)

 

Let's get started
No matters if you a pro or noob / Professional or Beginner to these languages with our tutorial you can easily understand and make this Analog Clock.

Step 5: Adding functionality (Java Script).
Adding this code will run the clock. The function will be called every second which will update the position of minute, hour and second hand to its position.

 

  var hr = document.getElementById("hr");
  var min = document.getElementById("min");
  var sec = document.getElementById("sec");
   
  setInterval(function () {
  var day = new Date();
   
  var secT = day.getSeconds() * 6;
  var minT = day.getMinutes() * 6;
  var hrT = day.getHours() * 30;
   
   
  sec.style.transform = `rotateZ(${secT}deg)`;
  min.style.transform = `rotateZ(${minT}deg)`;
  hr.style.transform = `rotateZ(${hrT + minT/12}deg)`;
   
  }, 1000);
